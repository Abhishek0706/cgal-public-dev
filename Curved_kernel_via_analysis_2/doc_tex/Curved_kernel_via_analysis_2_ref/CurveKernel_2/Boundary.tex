\begin{ccRefConcept}{CurveKernel_2::Boundary}

\ccDefinition

\ccRefName\ is a model of \ccc{RealEmbeddable} that is dense in \R. 
It is  used to represent upper or lower bounds for coordinates of curve points. 

\ccRefName\ is \ccc{ImplicitInteroperable} with 
\ccc{CurveKernel_2::Coordinate_1},
where \ccc{CurveKernel_2::Coordinate_1} is the coercion type. 

\ccRefines
\ccc{DefaultConstrctible}\\
\ccc{CopyConstructible}\\
\ccc{Assignable}\\
\ccc{FromIntConstructible}\\
\ccc{RealEmbeddable}\\
\ccc{ImplicitInteroperable} with \ccc{int}\\
\ccc{ImplicitInteroperable} with \ccc{CurveKernel_2::Coordinate_1}\\

\end{ccRefConcept}
