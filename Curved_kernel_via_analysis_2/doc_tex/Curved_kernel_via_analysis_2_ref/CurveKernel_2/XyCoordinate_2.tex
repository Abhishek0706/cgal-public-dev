\begin{ccRefConcept}{CurveKernel_2::XyCoordinate_2}

\ccDefinition

A instance of a model of this concept represents a finite point on a curve.
The model is required to be \ccc{LessThanComparable} with respect to a 
lexicographic order.

\ccRefines

\ccc{Assignable}\\
