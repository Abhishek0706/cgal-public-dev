\begin{ccRefClass}{Alpha_shape_face_base_2<Traits, 
	                                         Fb,
                                           ExactAlphaComparisonTag>}

\ccDefinition

The class \ccRefName\ is the default  model for the concept \ccc{AlphaShapeFace_2}.

\ccInclude{CGAL/Alpha_shape_face_base_2.h}

\ccParameters
The template parameter \ccc{Traits} has to be a model of \ccc{AlphaShapeTraits_2}.

The template parameter \ccc{Fb} has to be a model of \ccc{TriangulationFaceBase_2} (or \ccc{RegularTriangulationFaceBase_2})
if \ccClassTemplateName\ is intended to be used with an alpha-shape class based on a 
\ccc{Delaunay_triangulation_2} (or a \ccc{Regular_triangulation_2}).

The template parameter \ccc{ExactAlphaComparisonTag} is a tag that, when set to
\ccc{CGAL::Tag_true}, triggers exact comparisons between alpha values. See the description
provided in the documentation of \ccc{Alpha_shape_2} for more details. The default value is \ccc{CGAL::Tag_false}.


\ccIsModel
\ccc{AlphaShapeFace_2}

\ccInheritsFrom

\ccc{Fb}


\end{ccRefClass}
