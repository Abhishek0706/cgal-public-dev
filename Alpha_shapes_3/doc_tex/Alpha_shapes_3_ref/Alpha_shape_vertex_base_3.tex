\begin{ccRefClass}{Alpha_shape_vertex_base_3<Traits,Vb,ExactAlphaComparisonTag,WeightedTag>}

\ccDefinition

The class \ccRefName\ is the default  model for the concept
\ccc{AlphaShapeVertex_3}.

The class has four parameters : the traits class \ccc{Traits}
which provides the type for the points or the weighted points.
The second parameter \ccc{Vb} is a base class instantiated by default
with \ccc{CGAL::Triangulation_vertex_base_3<Traits>}. 
The third parameter \ccc{ExactAlphaComparisonTag} is a tag that, when set to
\ccc{CGAL::Tag_true}, triggers exact comparisons between alpha values. See the description
provided in the documentation of \ccc{Alpha_shape_3} for more details. The default value is \ccc{CGAL::Tag_false}.
The fourth parameter \ccc{WeightedTag} is used only if \ccc{ExactAlphaComparisonTag} is \ccc{CGAL::Tag_true}. It 
must be \ccc{CGAL::Tag_true} if the underlying triangulation of the alpha shape to be used is a regular triangulation
and \ccc{CGAL::Tag_false} otherwise. The default is \ccc{CGAL::Tag_false}.


\ccInclude{CGAL/Alpha_shape_vertex_base_3.h}


\ccIsModel
\ccc{AlphaShapeVertex_3}


\ccInheritsFrom

\ccc{Vb}

\end{ccRefClass}
