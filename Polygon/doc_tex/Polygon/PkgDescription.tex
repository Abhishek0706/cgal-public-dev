
\begin{ccPkgDescription}{2D Polygon\label{Pkg:Polygon2}}
\ccPkgHowToCiteCgal{cgal:gw-p2-12}
\ccPkgSummary{

This package provides a 2D polygon class and operations on sequences
of points, like bounding box, extremal points, signed area, simplicity
and convexity test, orientation, and point location.

The demo includes operations on polygons, such as computing a convex 
partition, and the straight skeleton.}
