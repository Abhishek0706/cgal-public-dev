\begin{ccRefClass}{Eigen_sparse_matrix<T>}

\ccDefinition

The class \ccc{Eigen_sparse_matrix} is a C++ wrapper around Eigen's matrix type \ccc{Eigen::SparseMatrix}
that represents general matrices, be they symmetric or not.
The version 3.1 (or greater) of \eigen\ must be available on the system.

\ccInclude{CGAL/Eigen_matrix.h}

\ccIsModel
Model of the \ccc{SparseLinearAlgebraTraits_d::Matrix} concept.

\ccParameters
\ccc{T}: Number type.

\ccTypes
\ccNestedType{EigenType}{The internal matrix type from \eigen.}

\begin{ccRefClass}{Eigen_sparse_symmetric_matrix<T>}

\ccDefinition

The class \ccc{Eigen_sparse_symmetric_matrix} is a C++ wrapper around \eigen's matrix type \ccc{Eigen::SparseMatrix}.

As the matrix is symmetric only the lower triangle part is stored.
