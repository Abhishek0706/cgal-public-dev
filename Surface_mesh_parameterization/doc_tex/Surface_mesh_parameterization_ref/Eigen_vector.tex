\begin{ccRefClass}{Eigen_vector<T>}

\ccDefinition
The class \ccc{Eigen_vector} is a C++ wrapper around \eigen's vector, which is a simple array of numbers.
The version 3.1 (or greater) of \eigen\ must be available on the system.

\ccInclude{CGAL/Eigen_vector.h}
