\begin{ccRefFunction}{solve_linear_program}

\ccInclude{CGAL/QP_functions.h}

This function solves a linear program, using some exact
Integral Domain \ccc{ET} for its computations. Various
options may be provided, see \ccc{Quadratic_program_options}.  

\ccFunction{template <LinearProgram, ET>
  Quadratic_program_solution<ET> solve_linear_program 
  (const LinearProgram& lp, const ET&,
  const Quadratic_program_options& options = Quadratic_program_options());}
{returns the solution of the linear program \ccc{lp}, solved
with exact number type \ccc{ET}.}
