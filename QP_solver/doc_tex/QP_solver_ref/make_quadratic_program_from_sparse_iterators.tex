\begin{ccRefFunction}{make_quadratic_program_from_sparse_iterators}

\ccInclude{CGAL/QP_models.h}

This template function creates an instance of 
\ccc{Quadratic_program_from_sparse_iterators<A_s_it, B_it, R_it, FL_it, L_it, FU_it, U_it, D_s_it, C_it>} from given iterators.
This function can be useful if the types of these
iterators are too complicated (or of too little interest for you) 
to write them down explicitly.



\ccFunction{template <
   A_s_it,   
   B_it,   
   R_it,   
   FL_it,  
   L_it,   
   FU_it, 
   U_it,   
   D_s_it,  
   C_it >  
Quadratic_program_from_sparse_iterators<A_s_it, B_it, R_it, FL_it, L_it, FU_it, U_it, D_s_it, C_it>
make_quadratic_program_from_sparse_iterators (
   int n, int m, 
   const A_s_it& a, 
   const B_it& b, 
   const R_it& r, 
   const FL_it& fl, 
   const L_it& l,
   const FU_it& fu, 
   const U_it& u, 
   const D_s_it& d, 
   const C_it& c, 
   std::iterator_traits<C_it>::value_type c0 = 
   std::iterator_traits<C_it>::value_type(0));}
{returns an instance of \ccc{Quadratic_program_from_sparse_iterators<A_s_it, B_it, R_it, FL_it, L_it, FU_it, U_it, D_s_it, C_it>}, constructed from the given iterators.}
