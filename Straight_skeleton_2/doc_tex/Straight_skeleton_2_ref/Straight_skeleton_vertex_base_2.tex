\ccDefinition

The class \ccRefName\ provides a model for the
\ccc{StraightSkeletonVertex_2} concept which is the vertex
type required by the \ccc{StraightSkeleton_2}
concept. The class \ccRefName\ has three template arguments: the first is 
the model of the \ccc{StraightSkeleton_2} concept (the vertex container), the second is a Point type, and the third is a model of the \ccc{SqrtFieldNumberType}, which is the numeric type used to represent the time of a vertex (a Euclidean distance).

This class can be used as a base class allowing users of the straight skeleton data structure to decorate a vertex with additional data. The concrete vertex class must be given in the \ccc{HalfedgeDSItems} template parameter of the instantiation of the \ccc{HalfedgeDS_default} class used as the model for the \ccc{Straight_skeleton_2} concept.
