\ccDefinition

The class \ccRefName\ converts a straight skeleton instantiated using certain traits into another straight skeleton instantiated using a different traits.

The first and second template parameters are the source and target straight skeleton types, resp.

The third template parameter \ccc{ItemsConverter} is a function object that must
model the concept \ccc{StraightSkeletonItemsConverter_2}. The default value of this parameter is \ccc{CGAL::Straight_skeleton_items_converter_2<SrcSs,TgtSs>}.

This conversion can be used to produce a straight skeleton with the fast
\ccc{Exact_predicates_inexact_constructions_kernel} but then input that skeleton
into \ccc{Polygon_offset_builder<Ss,Gt,Container>} instantiated with the slower \ccc{Exact_predicates_exact_constructions_kernel} obtaining only simple offset
polygons without paying the runtime overhead of exact constructions for the straight 
skeleton itself.
