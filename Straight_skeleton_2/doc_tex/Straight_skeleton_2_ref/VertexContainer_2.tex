\begin{ccRefConcept}{VertexContainer_2}

\ccHeading{Introduction}

A model for the \ccRefName\ concept defines the requirements for a resizable container of 2D points. It is used to output the offset polygons generated by the \ccc{Polygon_offset_builder_2<Ssds,Gt,Container>} class.

\ccTypes
  \ccNestedType{Point_2}{A 2D point type used to represent a vertex. Must be a model of the \ccc{Kernel::Point_2} concept}
\ccGlue  
  \ccNestedType{size_type}{A unsigned integral type that can represent the number of vertices in the container.}

\ccCreation
\ccCreationVariable{c}

\ccConstructor{VertexContainer_2();}{Default constructor}
\ccMethod{size_type size() const;}{Returns the number of vertices in the container.}
\ccMethod{void push_back( Point_2 const& v) const;}{Adds the vertex v to the container, resizing its capacity if required.}

\ccHasModels

\ccc{CGAL::Polygon_2}

Any \ccc{standard} \ccc{BackInsertionSequence}, such as \ccc{vector}, \ccc{list} or \ccc{deque}, with a \ccc{value\_type} being a model of the \ccc{Kernel::Point_2} concept.

\end{ccRefConcept} % StraightSkeleton_2
